# Head Soccer Pro

## Descripción
Videojuego arcade de fútbol 1 vs 1 desarrollado en Python utilizando la librería Pygame. El objetivo es marcar más goles que el rival antes de que se agote el tiempo.

## Requisitos de ejecución
- Python 3.x
- Librería Pygame (instalar mediante: `pip install pygame`)

## Instrucciones de ejecución
Para iniciar el juego, abre una terminal en la carpeta principal del proyecto y ejecuta el siguiente comando:
`python main.py`

## Controles del Juego
- **Jugador 1:** - A / D : Moverse de lado a lado
  - W : Saltar
  - ESPACIO : Patear el balón
- **Jugador 2:** - Flechas Izquierda / Derecha : Moverse de lado a lado
  - Flecha Arriba : Saltar
  - ENTER : Patear el balón
- **Controles Globales:** - ESC : Pausar / Despausar el partido
  - Ratón : Navegar por los menús y hacer clic en el icono del engranaje para Ajustes (Volumen y Pantalla Completa).
